Usage: sharp_daemons_setup.sh (-s | -r) [-p SHARP location dir]
   or: sharp_daemons_setup.sh -h
sharp_daemons_setup helps system administrator to manage SHARP daemons.
All management operations require root access.

Operation flags (must specify one for setup/remove):
	-s - Setup SHARP daemon (sharp_am)
	-r - Remove SHARP daemon (sharp_am)
	-h - Print this help and exit

Additional options:
	-p - Path to SHARP source installation directory (default: auto-detected)

Examples:
	Show this help message:
		/opt/mellanox/sharp/sbin/sharp_daemons_setup.sh -h
	Setup SHARP daemon with default path:
		/opt/mellanox/sharp/sbin/sharp_daemons_setup.sh -s
	Setup SHARP daemon with custom path:
		/opt/mellanox/sharp/sbin/sharp_daemons_setup.sh -s -p /path/to/sharp
	Remove SHARP daemon:
		/opt/mellanox/sharp/sbin/sharp_daemons_setup.sh -r

Note: sharp_am.service is NOT started automatically and does NOT support by default auto-restart.
Starting sharp_am.service resets SHARP trees and cleans allocated resources.
It could affect running applications.
